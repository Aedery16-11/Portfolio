Présentation du projet

Au cours de mon stage, j'ai travaillé sur un projet visant à créer une application de gestion des habilitations des employés, dont je détaillerai le fonctionnement.

Technologies utilisées

Ce projet a été principalement réalisé avec Flask, qui est un micro-framework permettant de créer des applications web simples.

Les données étaient conservées dans une base de données Google BigQuery, un service de Google Cloud Platform (GCP) conçu pour exécuter des requêtes SQL sur de vastes ensembles de données.

L'utilisation de librairies Python telles que pandas et pandasql m'a permis de filtrer et de formater les données avant de les afficher. "pandas" est une bibliothèque très utilisée pour la manipulation et l'analyse de données Excel, offrant un type de structure de données performant : le DataFrame. "pandasql", quant à lui, permet d'exécuter des requêtes SQL directement sur des DataFrames pandas, facilitant ainsi le traitement des données au sein de l'application. En plus de ces technologies, j'ai dû configurer l'environnement de développement en utilisant un environnement virtuel Python. Cela permet de gérer les dépendances/modules spécifiques au projet de manière isolée. La gestion des dépendances était effectuée à l'aide d'un fichier requirements.txt, qui liste toutes les librairies, frameworks nécessaires pour faire fonctionner l'application, et ce, dans la version dans laquelle elles ont été installées pour coder le projet.
